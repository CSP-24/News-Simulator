1. Install the requirements

   ```bash
   $ npm install
   ```
2. Make a copy of the example environment variables file

   On Linux systems: 
   ```bash
   $ cp .env.example .env
   ```
   On Windows:
   ```powershell
   $ copy .env.example .env
   ```
3. Add your [API key](https://beta.openai.com/account/api-keys) to the newly created `.env` file
4. Run the app

   ```bash
   $ npm run dev
   ```
